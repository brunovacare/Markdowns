# STYLUS
* Sass é escrito em ruby. Stylus é escrito em javascript. Assim, é muito mais simples criar um plugin para Stylus do que para Sass. 
* No Stylus, chaves, dois pontos e ponto-virgula são opcionais. 
* Resumidamente falando, o Stylus permite a escrita identica ao do Sass, mas tem mais funcionalidades. Assim, é fácil transformar um arquivo sass em stylus, mas o inverso nem sempre é possível.
